# 성적 평가 기준 (Grading Policy)

본 문서는 파이썬 및 웹 과정 학생들의 최종 성적 산출을 위한 공식 평가 비중을 정의합니다.

## 최종 성적 반영 비율 (Official Weights)

| 평가 항목 | 비율 (Weight) | 영문명 (English) | 비고 |
| :--- | :--- | :--- | :--- |
| **기말고사 / 프로젝트** | **40%** | Final Exam / Project | |
| **중간고사** | **20%** | Midterm Exam | |
| **과제** | **20%** | Assignments | 이메일/GitHub 폼 제출, 상호평가 결과 등 포함 |
| **수업 참여도** | **20%** | Class Participation | |
| **합계 (Total)** | **100%** | | |

> **참고**: 해당 기준은 전체 학기 최종 성적 산출 시 일괄 적용됩니다. 개별 과제 채점(예: 제출 점수 + 상호평가 점수 합산 로직)은 별도로 1.0 만점 체제로 처리되며, 학기 말에 모두 합산되어 위 '과제 (20%)' 비중에 편입됩니다.
