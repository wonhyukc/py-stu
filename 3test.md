# 3test.md (상호평가 다수결 로직 테스트 시나리오)

## 1. 다수결 점수 도출 (Majority Vote Calculation)
1. **명확한 다수결 존재**: 3명 중 2명 이상이 동일한 점수 배열을 주었을 때 해당 점수가 최종점수로 반환되는가?
2. **동점(Tie) 발생 시 처리**: 2명 채점자가 서로 다른 점수(1:1)를 주었을 때 0.8 가중치(학생에게 유리)에 맞춰 합이 더 큰 점수를 다수결로 선택하는가?

## 2. 평가자 참여 점수 (Evaluator Point Distribution)
1. **다수결 일치 시**: 올바르게 다수결에 속한 평가자가 배정 비율(`3.0 / 배정건수`) 만큼 점수를 획득하는가?
2. **소수 의견 감점**: 다수결과 엇갈린 소수 의견 평가자에게는 해당 건의 점수가 0점으로 처리되는가?

## 3. 최종 환산 로직 (Max Score Normalization)
1. 제출점수(가중치 0.8)와 평가점수(가중치 0.2)가 합산되어 최대 1.0 점으로 제한되는가?
2. 평가자가 총 2명만 배정되어 전부 맞췄을 때도 만점(0.2)을 받을 수 있도록 스케일링 되는가?
