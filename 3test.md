# Issue 33 Test Scenarios (분반 조회 및 필터링 적용)

## 단위 테스트 (Unit Tests)
1. **학생 명단 파싱 및 분반 매핑 성공 여부 파악**
   - **Target**: `bin/check_evaluations.py` (또는 분리된 헬퍼 함수)
   - **목표**: `input/students/py-students.md`를 읽어 `py` 분반 학생명단(학번) 리스트가 정상 추출되는지 확인.
   - **목표**: `input/students/wb-students.md`를 읽어 `wb` 분반 학생명단(학번) 리스트가 정상 추출되는지 확인.

2. **조건부 채점 필터링 동작 확인**
   - **Case**: 통합폼 Mock 데이터에 `py` 소속 평가자, 피평가자 응답과 `wb` 소속 응답이 섞여 있을 때.
   - **조건**: `--course py`로 실행 시, `py` 소속 응답 건만 추출되어 평가 가중치가 계산되는지 확인.

3. **CSV 컬럼 "분반" 추가 여부**
   - **Case**: 최종 CSV 파일의 첫 번째 열(`행[0]`)이 반드시 `분반`인지 확인.
   - **조건**: 출력 데이터의 모든 "분반" 값이 파라미터로 넘긴 값(e.g., `py`)과 일치하는지 확인.
