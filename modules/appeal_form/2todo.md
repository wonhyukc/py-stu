# 이의제기 설문지 실행 계획

## 1. 폼 템플릿 생성 및 설정
- [ ] 구글 폼 생성 (한/영 병용 표기 적용)
- [ ] 문항 구성 (학번, 이름, 트랙, 유형, 점수, 이의제기 내용)
- [ ] 데이터 유효성 검사 설정 (학번 10자리 2026 시작 정규식, 점수 숫자 제한)

## 2. 사전 채우기(Pre-filled) 링크 생성 자동화
- [ ] 구글 폼에서 '미리 채워진 링크 가져오기'를 통해 항목별 URL 쿼리 파라미터 구조 파악
- [ ] 학생 명부 데이터(구글 시트)를 읽어와 각 학생별 학번, 이름, 트랙 데이터를 추출하는 모듈 개발
- [ ] 각 데이터가 매핑된 고유 구글 폼 URL을 생성하는 파이썬 스크립트 작성
- [ ] 생성된 개별 설문지 링크를 각 학생의 이메일로 자동 발송하는 로직 구현
