# 이의제기 설문지 테스트 시나리오

## 1. 폼 유효성 검증 테스트
- [ ] 학번: `2026`으로 시작하지 않거나 길이가 10자리가 아닐 경우 제출 거부 및 에러 확인
- [ ] 점수: 숫자가 아닌 문자 입력 시 폼 자체에서 필터링되는지 확인

## 2. 사전 채우기(Pre-filled URL) 기능 테스트
- [ ] 파이썬 링크 생성 스크립트가 학생 명부 데이터와 누락 없이 매핑되어 URL 문자열을 반환하는지 단위 테스트 진행
- [ ] 생성된 링크 접속 시 폼의 '이름'과 '트랙', '학번' 필드에 올바른 데이터가 미리 입력되어 있는지 육안 확인
- [ ] 이메일 자동 발송 테스트: 테스트용 계정으로 발송하여 이메일 템플릿 내 링크가 정상 동작하는지 E2E 검증
